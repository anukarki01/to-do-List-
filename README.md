A modern, feature-rich task management application built with React.js, featuring a beautiful UI, dark/light mode, and comprehensive task management capabilities.



 **Features**

 
**Core Functionality**


✅ Add, Edit, Delete Tasks with rich metadata

✅ Task Status Management (Complete/Incomplete)

✅ Categories & Priorities (Personal, Work, Shopping, Health)

✅ Due Dates with calendar integration

✅ Form Validation with error handling

**Advanced Features**


 Dark/Light Mode with smooth transitions

 Search & Filter tasks by various criteria

 Progress Tracking with visual progress bar

 Calendar View for task scheduling

 Analytics Dashboard with charts and insights

 Local Storage for data persistence

 Drag & Drop task reordering

 Fully Responsive design



**UI/UX Enhancements**


 Smooth Animations with Framer Motion

 Modern Design with Tailwind CSS

 Real-time Updates

 Motivational Quotes widget

 Fast Performance with optimized rendering
